# Magic Touch 🪄

Este projeto foi desenvolvido como **Projeto Final do curso Projeto Desenvolve**.

A aplicação consiste em um **site fictício de e-commerce**, criado para praticar conceitos de desenvolvimento web utilizando **React** e consumo de **API externa**.

## 📌 Sobre o Projeto

A **Magic Touch** é uma loja online fictícia com estética feminina.  
O objetivo do projeto é simular a estrutura básica de um e-commerce, exibindo produtos obtidos de uma API e organizando a navegação em diferentes páginas.

O projeto demonstra conceitos importantes de desenvolvimento front-end, como:

- Estruturação de componentes em React
- Navegação entre páginas com React Router
- Consumo de API utilizando `fetch`
- Organização de código em pastas
- Layout responsivo simples
- Separação de responsabilidades entre componentes, páginas e serviços

## 🛍 Funcionalidades

- Menu de navegação funcional
- Página inicial com produtos em destaque
- Página de listagem de produtos
- Página de promoções
- Página de contato
- Consumo de produtos da API FakeStore

## 🧰 Tecnologias Utilizadas

- React
- Vite
- JavaScript
- HTML
- CSS
- React Router
- FakeStore API
